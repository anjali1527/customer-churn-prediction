# Customer Churn Prediction and Retention System

## Overview

Customer Churn Prediction and Retention System is a Machine Learning-based web application developed using Python and Streamlit. The system predicts whether a customer is likely to churn, analyzes risk levels, and provides retention strategies to help businesses reduce customer loss.

## Features

- Predict customer churn using Machine Learning
- Classify customers into risk levels
- Generate retention recommendations
- Interactive visualizations and insights
- CSV file upload support
- Automated report generation

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Plotly
- SHAP
- Joblib
